Introducción al Diseño Web

HTML (Hyper Text Markup Language)

HTML es un lenguaje de marcado de hiper texto, define la estructura y el contenido del documento. Esta basado en etiquetas y atributos. Puedes revisar todas las etiquetas en W3 Schools.

CSS (Cascading Style Sheets)

CSS es un lenguaje que da presentación a documentos HTML. Los estilos se pueden aplicar de 3 formas:

  1. En linea con el atributo style
  2. En la cabecera del documento con la etiqueta style
  3. Externamente desde una etiqueta link

Responsive Design

Es la técnica actual para hacer Diseño Web MultiDispositivo y consta de 3 principios:

  1. Maquetación Fluida
  2. Multimedia Flexible
  3. Uso de Mediaqueries

El Responsive Design tiene 2 enfoques:

  1. Mobile First
  2. Desktop First
